## 1. 实体关系

```
Company 1───N Financial（多期财务数据）
Company 1───N Comparable（已确认可比公司）
Company 1───N Valuation（估值结果，按方法拆分）
```

## 2. Company（公司）

**存储文件**：`data/companies.json`

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| `company_id` | string | PK, UUID | 自动生成 | 公司唯一 ID |
| `name` | string | 非空, ≤200 | — | 公司名称 |
| `industry` | string | 非空 | — | 行业分类 |
| `founded_year` | integer | 可选 | null | 成立年份 |
| `description` | string | 可选, ≤1000 | "" | 公司简介 |
| `is_listed` | boolean | — | false | 是否上市公司 |
| `stock_code` | string | 可选 | null | 股票代码（上市公司） |
| `industry_avg_pe` | number | 可选 | null | 行业平均 PE |
| `industry_growth_rate` | number | 可选 | null | 行业增速（%） |
| `has_valuation` | boolean | — | false | 是否已有估值 |
| `created_at` | string | ISO-8601 | 自动 | 创建时间 |
| `updated_at` | string | ISO-8601 | 自动 | 更新时间 |

## 3. Financial（财务数据）

**存储文件**：`data/financials.json`

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| `financial_id` | string | PK, UUID | 自动生成 | 记录 ID |
| `company_id` | string | FK → Company | — | 所属公司 |
| `period` | string | 非空 | — | 期间，如 "2025-FY" |
| `revenue` | number | ≥0 | — | 营收（万元） |
| `net_profit` | number | — | — | 净利润（万元，可为负） |
| `total_assets` | number | ≥0 | — | 总资产（万元） |
| `net_assets` | number | — | — | 净资产（万元） |
| `operating_cashflow` | number | — | — | 经营现金流（万元） |
| `eps` | number | 可选 | null | 每股净利润 |
| `stock_price` | number | 可选, ≥0 | null | 股价（上市公司） |
| `created_at` | string | ISO-8601 | 自动 | 导入时间 |

## 4. Comparable（可比公司关系）

**存储文件**：`data/comparables.json`

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| `record_id` | string | PK, UUID | 自动生成 | 记录 ID |
| `target_company_id` | string | FK → Company | — | 目标公司 ID |
| `comparable_company_id` | string | FK → Company | — | 可比公司 ID（上市公司） |
| `source` | string | enum | — | 来源：`ai_recommend` / `manual` |
| `reason` | string | 可选 | "" | 推荐理由 |
| `similarity_score` | number | 0~1 | null | AI 相似度分数 |
| `confirmed` | boolean | — | false | 用户是否已确认 |
| `confirmed_at` | string | ISO-8601 | null | 确认时间 |

## 5. Valuation（估值结果）

**存储文件**：`data/valuations.json`

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| `valuation_id` | string | PK, UUID | 自动生成 | 估值记录 ID |
| `company_id` | string | FK → Company | — | 目标公司 ID |
| `method_name` | string | enum | — | `pe_comparable` / `dcf` / `pb` / `ev_ebitda` |
| `valuation_low` | number | ≥0 | — | 估值下限（万元） |
| `valuation_mid` | number | ≥0 | — | 估值中位数（万元） |
| `valuation_high` | number | ≥0 | — | 估值上限（万元） |
| `params` | object | — | {} | 该方法的核心参数快照 |
| `calculation_detail` | string | — | "" | 计算过程说明 |
| `comparable_ids` | array | — | [] | 使用的可比公司 ID 列表 |
| `created_at` | string | ISO-8601 | 自动 | 计算时间 |

## 6. CRUD 接口签名

| 操作 | 方法签名 | 说明 |
|------|----------|------|
| 公司列表 | `get_companies(keyword?, industry?, page, page_size) → list` | 支持筛选 |
| 公司详情 | `get_company(company_id) → dict` | 含行业信息 |
| 导入财务 | `add_financial(company_id, data) → dict` | 同公司同期覆盖 |
| 获取财务 | `get_financials(company_id) → list` | 按期间排序 |
| AI推荐 | `recommend_comparables(company_id) → list` | 调用 AI 服务 |
| 确认可比 | `confirm_comparables(company_id, ids) → dict` | 批量确认 |
| 获取可比 | `get_comparables(company_id) → list` | 已确认的 |
| 运行估值 | `run_valuation(company_id, params?) → dict` | 多方法并行 |
| 估值详情 | `get_valuation_methods(company_id) → list` | 各方法明细 |
| 估值摘要 | `get_valuation_summary(company_id) → dict` | 综合+AI总结 |

## 7. 关键业务逻辑

| 逻辑 | 说明 |
|------|------|
| 财务数据覆盖 | 同一公司同一 period 重复导入时，覆盖旧记录 |
| 可比公司锁定 | 确认后 `confirmed=true`，估值计算仅使用已确认公司 |
| 估值触发前检查 | 运行估值前校验：目标公司有财务数据 + 至少 1 家已确认可比公司 |
| has_valuation 标记 | 首次估值完成后自动更新 Company.has_valuation = true |
| 重新估值 | 重新运行覆盖旧估值记录（同公司同方法） |
