## 1. SLO（服务级别目标）

| 指标 | 目标值 | 度量方式 |
|------|--------|----------|
| 公司列表响应时间 | ≤ 1s | 从请求到返回列表数据 |
| 估值计算响应时间 | ≤ 5s（3 种方法并行） | 从点击"运行估值"到全部方法返回 |
| AI 推荐响应时间 | ≤ 10s | 从点击"AI推荐"到返回可比公司列表 |
| 财务数据导入 | ≤ 2s | 从提交到入库完成 |
| 系统可用性 | ≥ 99%（教学版） | 工作时间内可访问 |
| PE 计算准确性 | 100%（公式级） | PE = 股价/EPS，无近似 |

## 2. 降级策略

| 场景 | 降级方式 | 用户感知 |
|------|----------|----------|
| AI 推荐不可用 | 降级到手动输入可比公司 | 提示"AI 推荐暂不可用" + 手动输入框 |
| LLM 总结不可用 | 仅展示数值结果，跳过 AI 总结 | 总结区域显示"AI 总结暂不可用" |
| 外部数据源不可用 | 使用本地模拟数据 | 标注"模拟数据，仅供参考" |

## 3. 数据约束

| 字段 | 约束 | 默认值 |
|------|------|--------|
| 公司名称 | 非空，≤200 字符 | — |
| 行业分类 | 非空 | — |
| 营收/净利润/总资产/净资产/现金流 | 数值型，单位万元 | — |
| 可比公司数量 | 1~10 家 | AI 推荐 4~6 家 |
| DCF 折现率 | 1%~30% | 10% |
| DCF 永续增长率 | 0%~15% | 5% |
| DCF 预测年数 | 3/5/7/10 | 5 |
| 每页公司数 | 1~100 | 20 |

## 4. 敏感数据保护

| 数据类型 | 教学版策略 | 生产版策略 |
|----------|-----------|-----------|
| 财务数据 | 本地 JSON 存储，无加密 | 数据库加密存储 + 访问日志 |
| 估值结果 | 本地存储 | 权限控制 + 审计 |
| AI 推荐内容 | 不缓存 API Key 到前端 | Token 鉴权 + 请求签名 |

## 5. 并发与容量

| 指标 | 教学版 | 生产版 |
|------|--------|--------|
| 并发用户 | 1~3 | 50+ |
| 公司数据量 | ≤100 家 | ≤10000 家 |
| 单次估值方法数 | 3~4 | 3~6 |
| JSON 文件大小 | ≤10 MB/文件 | 迁移至数据库 |
