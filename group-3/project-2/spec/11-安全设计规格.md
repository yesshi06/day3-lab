> 采用 **教学版 vs 生产版双轨对照**，明确当前实现与演进目标。

## 1. 认证与授权

| 维度 | 教学版（当前） | 生产版（演进） |
|------|--------------|--------------|
| 认证方式 | 无认证，直接访问 | JWT Token + SSO 集成 |
| 会话管理 | 无会话状态 | Token 有效期 + 刷新机制 |
| 角色权限 | 无区分，全功能开放 | 分析师/经理/管理员三级权限 |

## 2. 数据加密

| 维度 | 教学版（当前） | 生产版（演进） |
|------|--------------|--------------|
| 传输加密 | HTTP 明文 | HTTPS (TLS 1.2+) |
| 存储加密 | JSON 明文文件 | 数据库字段级 AES-256 加密 |
| API Key | 环境变量，不入库 | Vault 密钥管理 |

## 3. 接口安全

| 维度 | 教学版（当前） | 生产版（演进） |
|------|--------------|--------------|
| 限速 | 无限速 | 100 req/min/IP |
| CORS | 允许 localhost | 白名单域名 |
| 输入校验 | 基本类型校验（09 §13） | 严格 Schema 校验 + XSS 过滤 |
| 参数注入 | JSON 解析已自动防护 | 增加 SQL 注入防护（数据库演进后） |

## 4. AI/LLM 安全

| 维度 | 教学版（当前） | 生产版（演进） |
|------|--------------|--------------|
| Prompt 注入 | 固定 Prompt 模板，不暴露系统提示词 | Prompt 防护层 + 输出过滤 |
| 输出校验 | 前端展示时转义 | 结构化输出 + 内容审核 |
| API Key 保护 | 仅后端持有，不传前端 | 密钥轮换 + 调用审计 |
| 降级安全 | AI 不可用时降级到手动模式 | 熔断器 + 限流 |

## 5. 审计与日志

| 维度 | 教学版（当前） | 生产版（演进） |
|------|--------------|--------------|
| 操作日志 | Flask 控制台日志 | 结构化 JSON 日志 + ELK |
| 审计追踪 | traceId 贯穿请求链路 | traceId + userId + 操作时间 |
| 数据变更 | 无变更记录 | 财务数据变更审计表 |

## 6. STRIDE 威胁分析

| 威胁 | 风险 | 教学版应对 | 生产版应对 |
|------|------|-----------|-----------|
| Spoofing（欺骗） | 伪造请求 | 单机使用，风险低 | JWT 认证 |
| Tampering（篡改） | 篡改财务数据 | 文件权限控制 | 数据库事务 + 变更审计 |
| Repudiation（否认） | 操作不可追溯 | traceId 日志 | 完整审计日志 |
| Info Disclosure（泄露） | 财务数据泄露 | 本地存储 | 加密 + 权限 |
| DoS（拒绝服务） | 恶意请求 | 无防护 | 限速 + WAF |
| Elevation（提权） | 越权访问 | 无权限体系 | RBAC 三级权限 |

## 7. 部署安全

| 维度 | 教学版（当前） | 生产版（演进） |
|------|--------------|--------------|
| 运行环境 | 本地 Flask dev 服务器 | Gunicorn + Nginx 反向代理 |
| 端口暴露 | localhost:5000 | 仅 443 对外 |
| 依赖安全 | requirements.txt 固定版本 | 定期 Dependabot 扫描 |
| 密钥管理 | .env 文件 + .gitignore | Vault / 云密钥服务 |

## 8. 安全检查清单

- [ ] API Key 不出现在前端代码或日志中
- [ ] 所有用户输入经过类型校验（09 §13 参数校验规则）
- [ ] AI 推荐使用固定 Prompt 模板，不拼接用户原始输入到系统提示词
- [ ] JSON 文件不包含敏感密钥
- [ ] CORS 策略限制为开发环境域名
- [ ] traceId 贯穿全链路，可追溯
