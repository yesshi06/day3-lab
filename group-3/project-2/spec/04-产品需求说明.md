## 1. 愿景与目标

> 构建一套公司估值模型系统，帮助投研人员在 5 分钟内完成"选定目标公司 → 确认可比公司 → 多方法估值 → 查看结果"全链路，替代传统 Excel 手工估值流程。

| 03 痛点 | 04 目标 |
|---------|---------|
| 估值流程手工化，效率低下 | 全链路自动化：选公司→导数据→算估值→出结果，一站式完成 |
| 可比公司选择不稳定 | AI 推荐 4~6 家可比公司 + 用户确认机制，确保样本可控 |
| 多种估值方法难以并行对比 | PE/DCF/P/B 等多方法并行计算，统一视图展示 |
| 财务数据获取繁琐 | 教学版支持手动导入；生产版可接入外部数据 API |

## 2. 目标用户

| 角色 | 使用目标 | 频率 |
|------|----------|------|
| 投研分析师 | 对目标公司进行多方法估值分析，对比不同估值结果 | 每天数次 |
| 基金经理 | 查看估值结果摘要和 AI 推荐，辅助投资决策 | 每周数次 |
| 研究助理 | 录入目标公司财务数据，管理可比公司列表 | 每周数次 |

## 3. 核心业务场景

| 编号 | 场景名 | 用户路径 |
|------|--------|----------|
| SC-01 | 选定目标公司 | 用户进入公司列表 → 按名称/行业筛选 → 点击选定目标公司 → 进入公司详情页 |
| SC-02 | 查看公司详情与财务数据 | 用户在详情页查看基本信息 → 查看/导入财务三表 → 查看行业情况 |
| SC-03 | AI 推荐可比公司 | 用户点击"AI推荐" → 系统返回 4~6 家可比公司 + 推荐理由 → 用户勾选确认/手动增删 → 锁定样本 |
| SC-04 | 多方法估值计算 | 系统基于确认的可比公司 → 并行运行 PE/DCF/P/B 等方法 → 展示每种方法的参数明细和计算过程 |
| SC-05 | 估值结果综合展示 | 用户进入结果页 → 查看各方法估值区间 → 查看估值中枢 → 查看 AI 总结推荐 → 可返回调参重算 |

## 4. 产品规则

| 编号 | 规则 | 来源 |
|------|------|------|
| R-01 | 每次估值分析必须绑定一个目标公司 ID | 03 §3 "目标公司选择" |
| R-02 | 可比公司数量 1~10 家，AI 默认推荐 4~6 家 | 03 §3 "可比公司推荐" |
| R-03 | PE 计算公式：PE = 股价 / 每股净利润（EPS）；估值 = 可比公司 PE 中位数 × 目标公司净利润 | 会议讨论 |
| R-04 | 所有估值方法共用同一组可比公司样本，不同方法不同结果均展示 | 会议讨论 |
| R-05 | 财务三表字段：营收、净利润、总资产、净资产、经营现金流（最少） | 03 §3 "财务三表导入" |
| R-06 | AI 推荐不可用时，用户可手动输入可比公司，系统不中断 | 03 §5 风险应对 |
| R-07 | DCF 法支持用户调参（折现率、增速假设）后重新计算 | 会议讨论 |

## 5. 能力与优先级

| 子能力 | 优先级 | 对应场景 |
|--------|--------|----------|
| 公司列表搜索与筛选 | **P0** | SC-01 |
| 公司详情与财务三表展示/导入 | **P0** | SC-02 |
| 可比公司 AI 推荐与用户确认 | **P0** | SC-03 |
| PE 可比法估值计算 | **P0** | SC-04 |
| DCF 现金流折现法估值计算 | **P0** | SC-04 |
| P/B 市净率法估值计算 | **P0** | SC-04 |
| 估值结果综合展示与 AI 总结 | **P0** | SC-05 |
| EV/EBITDA 估值计算 | **P1** | SC-04 |
| 估值报告导出（JSON/文本） | **P1** | SC-05 |
| 可比公司历史股价趋势图 | **P1** | SC-02, SC-04 |

## 6. 验收与成功标准（PRD 层）

| 标准 | 度量 |
|------|------|
| 主路径可走通 | 用户在 5 分钟内完成"选定目标公司 → 确认可比公司 → 查看估值结果"全链路 |
| 估值方法覆盖 | 至少 3 种估值方法（PE/DCF/P/B）并行输出结果 |
| 估值结果可用 | 各方法估值结果均附带参数明细与计算过程，可供人工校验 |
| 可演示 | 模拟数据 ≥ 3 家公司下，主路径 + 估值结果 **5 分钟内**演示完毕 |
| 降级可用 | AI 推荐不可用时，手动输入可比公司后估值流程不中断 |
