## 1. 页面总览

| 页面 | 路由 | 对应 US | 说明 |
|------|------|---------|------|
| 公司筛选页 | `/companies` | US-001 | 搜索、筛选、选定目标公司 |
| 公司详情页 | `/companies/:id` | US-002 | 基本信息 + 财务三表 + 行业情况 |
| 可比公司页 | `/companies/:id/comparables` | US-003 | AI 推荐 + 用户确认 |

| 估值结果页 | `/companies/:id/summary` | US-005 | 综合结果 + AI 推荐 |

## 2. 公司筛选页（US-001）

### 2.1 布局

```
┌──────────────────────────────────┐
│  搜索栏: [关键字输入] [行业下拉]  │
├──────────────────────────────────┤
│  公司卡片列表（网格/列表切换）     │
│  ┌────────┐ ┌────────┐          │
│  │ 公司A  │ │ 公司B  │ ...      │
│  │ 行业   │ │ 行业   │          │
│  │ ★已估值│ │        │          │
│  └────────┘ └────────┘          │
├──────────────────────────────────┤
│  分页: < 1 2 3 >                │
└──────────────────────────────────┘
```

### 2.2 组件行为

| 状态 | 展示 |
|------|------|
| 空状态（无公司数据） | 居中提示"暂无公司数据" |
| 搜索无结果 | 提示"未找到匹配公司，请调整关键字" |
| 正常列表 | 公司卡片，已有估值的显示 ★ 标识 |

### 2.3 核心 State

| 变量 | 类型 | 说明 |
|------|------|------|
| `keyword` | string | 搜索关键字 |
| `industry` | string | 行业筛选值 |
| `companies` | array | 公司列表 |
| `loading` | boolean | 加载状态 |
| `page` / `total` | number | 分页 |

### 4.2 组件行为

| 状态 | 展示 |
|------|------|
| 未推荐 | 仅显示 AI 推荐按钮 |
| 推荐中 | 加载动画 + "正在分析中..." |
| AI 不可用 | 警告提示 + 手动添加入口 |
| 推荐完成 | 勾选列表，默认全选 |
| 确认后 | 跳转至估值分析页 |

## 5. 估值分析页（US-004）

### 5.1 布局

```
┌──────────────────────────────────┐
│  可比公司数据总览表                │
│  公司 | 股价 | PE | P/B | 净利润  │
├──────────────────────────────────┤
│  估值方法卡片（横向排列）          │
│  ┌─────────┐ ┌─────────┐ ┌──── │
│  │ PE可比法 │ │ DCF法   │ │ P/B │
│  │ 参数明细 │ │ 可调参  │ │     │
│  │ 计算过程 │ │ [重算]  │ │     │
│  │ 估值区间 │ │ 估值区间│ │     │
│  └─────────┘ └─────────┘ └──── │
├──────────────────────────────────┤
│  [查看综合结果 →]                 │
└──────────────────────────────────┘
```

### 5.2 DCF 调参交互

| 参数 | 输入控件 | 范围 | 默认值 |
|------|----------|------|--------|
| 折现率 | 滑块+输入框 | 1%~30% | 10% |
| 永续增长率 | 滑块+输入框 | 0%~15% | 5% |
| 预测年数 | 下拉 | 3/5/7/10 | 5 |

修改参数后点击"重新计算"，仅重算 DCF 方法。

## 7. 错误码前端提示映射

| error.code | 前端提示 |
|------------|----------|
| `COMPANY_NOT_FOUND` | "公司不存在，请返回列表重新选择" |
| `INVALID_FINANCIAL_DATA` | "财务数据格式有误，请检查后重新导入" |
| `NO_COMPARABLES` | "请先选择并确认可比公司" |
| `AI_UNAVAILABLE` | "AI 推荐暂不可用，请手动添加可比公司" |
| `VALUATION_NOT_FOUND` | "尚未运行估值，请先执行估值计算" |
| `INVALID_PARAMS` | "参数超出合理范围，请调整后重试" |
| `INTERNAL_ERROR` | "系统异常，请稍后重试" |
