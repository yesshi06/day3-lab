## 估值算法详细规格

### PE可比法
```
输入：目标公司净利润、可比公司列表(含股价+EPS)
计算：
  - 各可比PE = 股价/EPS（排除EPS≤0的异常值）
  - PE中位数 = median(PE列表)
  - 估值 = PE中位数 × 目标公司净利润
  - 区间 = [PE_min×净利润, PE_max×净利润]
验证点：中位数 vs 均值选择、异常值处理、负利润处理
```

### DCF现金流折现法
```
输入：基期现金流、折现率r(10%)、增长率g(5%)、预测年数n(5)
计算：
  - 各年预测CF_t = 基期CF × (1+g)^t
  - 各年折现值PV_t = CF_t / (1+r)^t
  - 终值TV = CF_n × (1+g_perpetual) / (r - g_perpetual)
  - 终值折现PV_TV = TV / (1+r)^n
  - 估值 = Σ PV_t + PV_TV
验证点：r > g_perpetual 约束、折现因子精度、终值占比合理性
```

### P/B市净率法
```
输入：目标公司净资产、可比公司列表(股价+每股净资产)
计算：
  - 各可比P/B = 股价/每股净资产（排除净资产≤0的异常值）
  - P/B中位数 = median(P/B列表)
  - 估值 = P/B中位数 × 目标公司净资产
验证点：负净资产处理、行业P/B合理范围
```
